# small-projects
## mandelbrot.c
An interactive visualizer for the mandelbrot fractal. Default keybinds:
| Key | Functionality  |
| --- | -------------- |
| W   | Move up        |
| A   | Move left      |
| R   | Move down      |
| S   | Move right     |
| P   | Zoom in        |
| T   | Zoom out       |
| G   | Increase depth |
| D   | Decrease depth |
